Notes

Fonctionnalités :
• Nouvelle note : Cliquez sur "Nouvelle note"
• Éditer : Cliquez sur une note pour l'ouvrir et modifier
• Bloc code : Ajoutez du code dans le champ dédié ; il sera copiable en un clic
• Couleur : Choisissez une couleur pour organiser visuellement vos notes
• Supprimer : Bouton dans le modal d'édition
