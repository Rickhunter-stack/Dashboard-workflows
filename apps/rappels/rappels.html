🔔 Rappels

🎯 Fonctionnalités:
• Ajouter des rappels: Cliquez sur "Nouveau rappel"
• Éditer: Cliquez sur le texte d'un rappel pour le modifier
• Date et heure: Cliquez sur l'icône 📅 pour paramétrer la date du rappel
• Alertes: Configurez quand vous voulez être alerté (1h, 1j, 1 semaine avant)
• Priorités: Utilisez les boutons colorés (● rouge = urgent, ● orange = important, ● vert = normal)
• Supprimer: Cliquez sur l'icône 🗑️ pour supprimer un rappel
• Filtrer: Utilisez la barre de recherche pour filtrer les rappels

⚠️ Notifications:
Les alertes s'affichent dans l'interface lorsque vous ouvrez l'application.
Les rappels passés apparaissent en rouge pour attirer votre attention.
