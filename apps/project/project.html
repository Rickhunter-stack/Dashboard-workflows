Gestion de projet

🎯 Fonctionnalités:
• Ajouter des projets: Cliquez sur "Nouveau Projet"
• Ajouter des tâches: Cliquez sur le bouton + dans une cellule vide
• Éditer: Cliquez sur le texte d'un projet ou d'une tâche pour le modifier
• Déplacer horizontalement: Glissez-déposez les tâches entre les phases (sur la même ligne)
• Priorités: 3 boutons de couleur (rouge = urgent, orange = important, vert = normal)
• Dates: Définissez des dates de début et fin pour chaque tâche
• Filtrer: Utilisez la barre de recherche pour filtrer par nom
• Supprimer: Cliquez sur l'icône 🗑️ pour supprimer une tâche ou un projet

📅 Organisation:
Les projets sont organisés en lignes horizontales.
Chaque projet peut avoir une tâche dans chaque phase.
Les tâches peuvent se déplacer horizontalement entre les phases mais restent sur leur ligne de projet.

💡 Astuce: Sur mobile et tablette, faites défiler horizontalement pour voir toutes les phases.
