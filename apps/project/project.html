Gestion de projet

Comment utiliser

🎯 Fonctionnalités:
• Ajouter des projets: Bouton "Nouveau Projet"
• Ajouter des tâches: Bouton + dans une cellule vide
• Éditer: Cliquez sur le texte d'un projet ou d'une tâche
• Déplacer: Drag & drop horizontal dans les phases

📅 Dates: Définissez les dates début/fin dans chaque tâche.
🔎 Filtrer: Utilisez la barre de recherche.
