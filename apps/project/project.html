Projets

Comment utiliser

• Ajouter un projet : Nouveau Projet
• Ajouter une tâche : bouton + dans une case
• Déplacer : drag & drop horizontal

Éditer : cliquez sur le texte d’un projet ou d’une tâche.
Priorités : points rouge/orange/vert.
Dates : définissez début/fin dans chaque tâche.
Filtrer : utilisez la barre de recherche.
