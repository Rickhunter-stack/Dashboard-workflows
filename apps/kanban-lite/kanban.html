Kanban-lite

🎯 Fonctionnalités:
• Ajouter des cartes: Cliquez sur "Nouvelle carte" ou le bouton + dans une colonne
• Éditer: Cliquez sur le titre d'une carte ou colonne pour le modifier
• Notes: Ajoutez des détails dans la zone de texte de chaque carte
• Réorganiser: Glissez-déposez les cartes entre les colonnes
• Filtrer: Utilisez la barre de recherche pour filtrer par titre ou note
• Supprimer: Cliquez sur l'icône 🗑️ pour supprimer une carte

⌨️ Raccourcis clavier:
• Tab / Shift+Tab: Naviguer entre les éléments
• Enter: Commencer/valider l'édition d'un titre
• Esc: Annuler l'édition en cours
