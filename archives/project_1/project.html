🗺️ Do It Fix It

🎯 Fonctionnalités:
• Ajouter des tâches: Cliquez sur "Nouvelle tâche" ou le bouton + dans une phase
• Éditer: Cliquez sur le texte d'une tâche pour le modifier
• Réorganiser: Glissez-déposez les tâches entre les phases
• Priorités: 3 boutons de couleur (rouge = urgent, orange = important, vert = normal)
• Dates: Définissez des dates de début et fin pour chaque tâche
• Filtrer: Utilisez la barre de recherche pour filtrer par nom
• Supprimer: Cliquez sur l'icône 🗑️ pour supprimer une tâche

📅 Phases du projet:
Les tâches sont organisées en 4 phases : Planification, Développement, Test, et Déploiement.
Vous pouvez déplacer les tâches entre les phases pour suivre leur progression.
